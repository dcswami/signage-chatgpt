# Kiosk Layout Samples

Open `kiosk-layout-options.html` in a browser to review three front-end kiosk layout options, each shown in Available, Busy, and Buffer/Warning states:

- Classic Institutional
- Event Formal
- Custom Background

The Classic Institutional and Event Formal samples use the Modern Minimal-style header and the Event Formal-style footer. Header and footer areas are set to 10% of the kiosk frame height. The Custom Background sample uses `../assets/backgrounds/background.png`. Upcoming events include dates and use smaller list typography. The samples use the default logo from `../assets/branding/aksharderi-small2.png` and placeholder booking QR visuals. The production application should replace the placeholder QR visuals with generated room booking QR codes.
